# Golang yEncode (yenc) module

## `import "gopkg.in/yenc.v0"`

A Golang port of [yencode 0.46](https://sourceforge.net/projects/yencode/files/yencode/0.46/) that encodes and decodes binary data for Usenet (NNTP) transfer.
